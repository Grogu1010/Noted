# Noted

A calm, local-first notes app designed for GitHub Pages.

## Publish on GitHub Pages

1. Upload the four website files from this package to the root of your repository.
2. Open **Settings → Pages** in GitHub.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select `main`, choose `/ (root)`, and save.

Your notes remain stored locally in each visitor's browser. No account is required.
