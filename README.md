# Noted

A calm, local-first notes app designed to feel complete without accounts, collaboration, or AI.

## GitHub Pages

Everything needed by the website sits directly in the repository root—there are no `src`, `public`, or build folders to upload.

If Pages is not already enabled, open **Settings → Pages**, choose **Deploy from a branch**, select `main` and `/ (root)`, then save.

Notes, folders, styling, and password verifiers remain on each visitor’s device. No account is required.
